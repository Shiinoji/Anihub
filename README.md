AniHub is a modern anime tracking application powered by the AniList API. It allows users to discover anime, view detailed information, and manage their personal watchlists with ease. Users can track their watching progress, organize anime into custom lists, and stay updated with their favorite series through a clean and intuitive interface. Built with Android and designed for a smooth user experience, AniHub provides a convenient way for anime fans to keep their anime journey organized in one place.
