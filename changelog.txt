AniHub Version 1.1 Changelog

New Features:
- Immersive Character Detail Screen: Explore deep metadata including biographies, appearances, and character stats.
- Integrated Notification Center: A dedicated place to view airing updates, new anime releases, and app news.
- Personalized Discovery: Added "Recommended for You" and "Explore More" sections to the Home screen.
- Advanced AniList Settings: Added Title Language (Romaji/English/Native) and Score Format (100/10.0/10/5/3) preferences.
- Shimmer Skeleton Loading: Replaced progress bars with modern shimmer effects for a smoother loading experience.

Fixes & Improvements:
- Refined UI: Added blur backgrounds to detail screens for a premium feel.
- Layout Fix: Corrected status bar padding issues where headers were too high on the screen.
- Icon Assets: Fixed Android resource linking errors with custom XML drawables.
- Watchlist Optimization: Simplified the watchlist and history cards for better performance and a cleaner look.
- Performance: Optimized background episode tracking (WorkManager) to be more battery-efficient.
- Navigation: Standardized the top headers and back navigation across all discovery screens.
